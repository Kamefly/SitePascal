Il existe plusieurs types de boucles :

  • Les boucles FOR
  • Les boucles WHILE
  • Les boucles REPEAT

Pour faire simple, disons qu'une boucle est une machine à laver. Elle continuera à tourner tant que son objectif n'est pas remplis.

Par exemple, l'une de ces machines ne s'arretera pas tant qu'elle n'aura pas fait le linge avec un certain paramètre.

C'est ici la boucle FOR qui est décrite, elle executera ce qui est à l'intérieur de sa boucle avec, par exemple,

une variable avec une valeur différente.


Une autre machine à laver ne s'arretera que après avoir effectuer un nombre de tour prédéfini.

Là, c'est la boucle WHILE qui est décrite et le nombre de tour est une variable. Tant que cette variable n'aura pas atteind

une certaine valeur
