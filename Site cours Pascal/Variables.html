Nous allons ici voir ce qu'est une Variable


Une variable est une donnée qui pourra changer au cours d'un programme, d'où le nom "variable".

Il existe plusieurs types de variables :

  • Les entiers ( integer )
  • Les nombres réels ( real )
  • Les charactères ( char)
  • Les chaines de caractères ( string )
  • Les booléens ( bool )

En pascal, une variable est notée de cette façon :


VAR

nom_de_la_variable:integer

ou

nom_de_la_variable:real

ou

nom_de_la_variable:char

ou

nom_de_la_variable:string

ou

nom_de_la_variable:bool
